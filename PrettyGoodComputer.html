Pretty Good Computer

Instructions

  • zn
    • Zeros a memory location n
  • in
    • Increments a memory location n
  • jn!m)l
    • Compares memory locations n and m for equality. If n and m are equal, proceeds to the next line, otherwise goes to line l

Example Program

z1
z2
i0
i0
i0
i1
j1!0)6
